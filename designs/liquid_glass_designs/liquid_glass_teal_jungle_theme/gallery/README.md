# Liquid Glass Teal Jungle Theme - Gallery

## Design Preview

This folder contains screenshots and previews of the **Liquid Glass Teal Jungle Theme** login page design.

### Theme Features

- **Glass Morphism Effect**: Frosted glass login card with semi-transparent background
- **Color Palette**: Light teal background with deep teal accents
- **Nature Elements**: Jungle tree silhouettes in the background creating depth
- **Wildlife Silhouettes**: Elephants, deer, and other jungle animals at the bottom
- **Modern UI**: Clean, minimalist form design with intuitive input fields
- **Social Integration**: Google, Meta, and VK login options

### Theme Files

- `liquid_glass_teal_jungle_theme.html` - Main login page HTML
- `google_logo.png` - Google social login icon
- `meta_logo.png` - Meta/Facebook social login icon
- `vk_logo.png` - VK/VKontakte social login icon

### Design Overview

The login page features a serene jungle landscape with layered silhouettes of trees and animals, creating a sense of depth and natural beauty. The glass-morphism login card sits prominently in the center, blending modern design with nature-inspired aesthetics.

### Design Inspiration

This design combines the trendy liquid glass morphism effect with a tranquil jungle theme, creating a unique and visually appealing login experience.
